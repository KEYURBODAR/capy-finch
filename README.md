# capy-finch

Finch — a clean, minimal Twitter/X front-end. Built on the Nitter codebase with extensive UI, performance, and feature additions.

## Features

- Following & Lists with local-first collections
- Articles extraction and reading
- Attention feed algorithm
- RSS feeds for all surfaces
- Export/import data bundles
- Privacy-focused (no JavaScript tracking)
- Fast SSR with Nim

## Setup

See `nitter.example.conf` for configuration.
